# skelton70

My Ruby on Rails 7.0 boilerplate.

* Ruby 3.2.2
* Ruby on Rails 7.0.5
  * `rails new . -T`-c=bootstrap
* Haml
  * with [Hamlit](https://github.com/k0kubun/hamlit)
* FontAwesome
* Active Record Session Store
* Config
* dotenv
* Whenever
* Awesome Print
* Annotate
* rack-dev-mark
* rack-mini-profiler
* RSpec
* DatabaseRewinder
* factory_bot
* Guard
* SimpleCov
* Brakeman
* rails_best_practices
